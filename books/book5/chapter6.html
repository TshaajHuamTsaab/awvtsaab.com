鹰鸽博弈：竞争与合作的进化平衡

1. 什么是进化稳定策略（ESS）

定义：一个策略是进化稳定的，如果整个种群的绝大多数个体都采用该策略，那么任何稀少的、不同的突变策略都无法入侵并取代它。简单来说，ESS是一种进化上的“纳什均衡”，一旦种群达到这种状态，就不会有偏离的动力。

2. 经典模型：鹰鸽博弈

场景：个体为了争夺稀缺资源（食物、领地、配偶）而竞争，拥有两种策略：

  • 鹰派：攻击性强，战斗至一方逃走或受伤。
  • 鸽派：仅进行仪式性炫耀，如遇攻击则逃避。

收益设定：

  • 获胜收益 = +50
  • 失败代价 = -100
  • 平局收益 = 0
  • 时间/精力消耗 = -10

平均收益分析：

  • 鹰 vs 鹰：(-100 + 50)/2 = -25
  • 鹰 vs 鸽：鹰 +50，鸽 0
  • 鸽 vs 鸽：(+50 + 0 -10)/2 = +20
  • 鸽 vs 鹰：同上，鸽收益 0

3. 寻找进化稳定策略

全鸽派？鹰派会迅速扩散 → 不稳定
全鹰派？鸽派能避免损失 → 不稳定
混合策略？达到稳定比例时，鹰派与鸽派的平均收益相等 → 稳定（ESS）

计算：设鹰派比例为 p，鸽派比例为 1-p

鹰派期望收益 = -25p + 50(1-p)
鸽派期望收益 = 0*p + 20(1-p)
令两者相等解得 p ≈ 5/8

结论：约 5/8 的鹰派与 3/8 的鸽派构成 ESS。

4. 更深层启示与现实例证

  • 平衡的必然性：纯侵略或纯和平在进化中不稳定，种群需要勇士与外交家。
  • 条件性策略才是终极 ESS：
    • 种群层面：固定比例的鹰/鸽个体
    • 个体层面：条件性策略，例如 bourgeois 策略（资源主人为鹰，入侵者为鸽）
    • 评估对手：根据体型、实力、过往表现决定战还是退
  • 现实例证：
    • 动物求偶与争斗：雄鹿通过比较角的大小进行仪式性较量，弱者退却
    • 人类商业竞争：公司并非总价格战，有时形成默契共存，但核心利益受侵时反击

总结

进化稳定策略理论说明，生物行为的多样性（竞争与合作、侵略与和平）不是偶然，而是在特定成本与收益结构下，通过进化博弈自发形成的平衡状态。鹰鸽博弈展示了为何合作与竞争会长期共存，因为它们彼此制约，形成动态的进化“纳什均衡”。
